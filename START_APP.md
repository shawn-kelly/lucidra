## 🔧 Troubleshooting

**Port already in use:**
```bash
netstat -tlnp | grep :8000
kill -9 [PID]
```

**Permission denied:**
```bash
chmod +x serve.py
```

**Browser doesn't auto-open:**
Manually visit: `http://localhost:8000`
