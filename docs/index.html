Bad Strategy Detection

🚫 Fluff

Superficial restatement of the obvious

Contains vague language without substance
Uses trendy buzzwords instead of clear thinking
Restates goals as strategy

Ignores obvious problems
Avoids making difficult trade-offs
Lists everything as equally important
